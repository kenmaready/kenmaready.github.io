Introduction

pandas is a Python package providing fast, flexible, and expressive data structures designed to make working with “relational” or “labeled” data both easy and intuitive. It aims to be the fundamental high-level building block for doing practical, real world data analysis in Python. Additionally, it has the broader goal of becoming the most powerful and flexible open source data analysis / manipulation tool available in any language. It is already well on its way toward this goal.

pandas is well suited for many different kinds of data:

  • Tabular data with heterogeneously-typed columns, as in an SQL table or Excel spreadsheet
  • Ordered and unordered (not necessarily fixed-frequency) time series data
  • Arbitrary matrix data (homogeneously typed or heterogeneous) with row and column labels
  • Data imported from a .csv (comma-separated values) or .xslx (excel) file
  • Any other form of observational / statistical data sets. The data actually need not be labeled at all to be placed into a pandas data structure

Data Structures

The two primary data structures of pandas, Series (1-dimensional) and DataFrame (2-dimensional), handle the vast majority of typical use cases in finance, statistics, social science, and many areas of engineering. For R users, DataFrame provides everything that R’s data.frame provides and much more. pandas is built on top of NumPy and is intended to integrate well within a scientific computing environment with many other 3rd party libraries.

We’ll start with a quick, non-comprehensive overview of the fundamental data structures in pandas to get you started. The fundamental behavior about data types, indexing, and axis labeling / alignment apply across all of the objects. To get started, import NumPy and load pandas into your namespace:

>> import numpy as np

>> import pandas as pd

We’ll give a brief intro to the data structures, then consider all of the broad categories of functionality and methods in separate sections.

Series

A Series is a one-dimensional labeled array capable of holding any data type (integers, strings, floating point numbers, Python objects, etc.). The axis labels are collectively referred to as the index. The basic method to create a Series is to call:

>> s = pd.Series(data, index=index)

where s is the variable name for the new Series, data can be any one of a number of python data structures (lists, numpy array, scalar, etc.), and index is a list of index labels.

DataFrames

A DataFrame is a 2-dimensional labeled data structure with columns of potentially different types. You can think of it like a spreadsheet or SQL table, or a dict of Series objects. It is generally the most commonly used pandas object. A DataFrame can be created out of many types of data collections, including lists, series, arrays, etc. Pandas provides easy means to create DataFrames from data in other file (e.g., a csv file or an excel file).

A basic method to create a DataFrame is:

>> data2 = [{'a': 1, 'b': 2}, {'a': 5, 'b': 10, 'c': 20}]

>> df = pd.Series(data2, index=index)

The resulting DataFrame (df) would be a table of 3 columns ('a', 'b' and 'c') with the first row (titled '0' by default) containing the values 1, 2 and NaN (for 'Not a Number'), and the second row (titled '1') containing the values 5, 10 and 20.

Creating DataFrames From Files

As mentioned above in the Data Structures section, pandas provides easy methods for importing data from outside sources directly into a DataFrame structure. Often, data will be in a .csv (comma-separated values) or an .xls (excel) file. For these, the pandas.read_csv() and pandas.read_excel() methods make it easy to import the data directly into a DataFrame structure.

The call is fairly straightforward -- for example:

>> df = pd.read_csv('mydatafile.csv', index_col='Date')

This example reads all the data in from a .csv file called 'mydatafile.csv' and puts it into a DataFrame named df, using the "Date" column from the file as the index for the new DataFrame. The method is very flexible and provides a lot of useful optional parameters so that you can select only certain columns, certain rows, rename the columns, set the value you want in your DataFrame to replace NaN values, etc.

Saving Data to Files

Pandas also makes it easy to export data from a DataFrame to filetypes commonly used for data. For example, the flipside of the .from_csv() and .from_excel() methods are to_csv() and to_excel(). As a simple example, in order to save the DataFrame df that we created from a csv file above:

>> df = pd.to_excel('myexcelfile.xslx')

Visualizing Data

In order to allow for easy visualization, pandas 'plays nice with' matplotlib. For many of the matplotlib methods, pandas provides a similar method (typically using the same naming conventions), with the pandas method being a 'wrapper' providing enhanced functionality with pandas data structures. For example, in order to plot the data in all the columns of a DataFrame in a line plot, one only has to use the pandas version of the .plot() function:

>> import matplotlib.pyplot as plt


>> # create DataFrame df with time series data

>> # and columns 'A', 'B', 'C' and 'D'

>> df.plot()

>> plt.show()
